# Приложение Loyalty

## Установить зависимости

Необходимо установить [just](https://github.com/casey/just)
```
uv pip install -e ".[dev]"
```

## Запустить проект
```
just dev
```
Чтобы остановить приложение:
```
just down
```
Чтобы очистить данные приложения:
```
just clear
```
Чтобы запустить линтеры:
```
just lint
```

## Тесты
Запуск e2e тестов
```
just e2e
```
